# runteq_app_listener

## サービス概要
アプリ名：Listener（リスナー）
本アプリの詳細：
本アプリは、学習者がテキストの収録されている音声を使用して、音読練習・オーバーラッピングやシャドーイング練習を行うことができるアプリです。
リピート機能、0.７〜1.5倍速まで速度変更、バックグラウンド再生など、学習者の好みに合わせて調整することができます。
 

## このサービスへの思い・作りたい理由
大きく分けて２つ。１つ目は、私自身の英語学習の経験から、２つ目は現職での実感からです。
私は約６年間、独学で英語を学習してきました。初めは全く歯が立たなかったTOEICも300点から940まで伸び、現在はそれを仕事として英語を使っています。
これまでの学習過程の中で強く実感したのは、**「英語学習は復習が命である」**ということです。私は「小さな復習の積み重ねこそがピラミッドの土台となり、その上に英語力が着実に積み上がっていく」と確信しています。英語を勉強以外で使用しない日本人が一定の英語力を身につけるには、「復習」しかないと思っています。

また現在、英語講師として働く中で、生徒と面談して学習状況を確認する機会がよくあります。そうした面談を重ねる中で気づいたのは、多くの生徒が自宅学習を「単語学習」に偏らせており、リーディングやリスニングに取り組む生徒はごくわずかであるという現状です。

こうした現状を受け、私はより多くの生徒が音声を活用し、音読やシャドーイングを通じて学習を深められるようなアプリを開発したいと考えるようになりました。

## ユーザー層について
現職の英会話スクールの生徒のみ対象（小学生から高校生まで） 
【その理由】：会社の幹部の方に相談したところ、スクールに通う生徒のみという条件でアプリ開発の承諾をいただきました。

## サービスの利用イメージ
スクールに通う生徒が自宅で本アプリを用いて、音声を用いた英語長文の音読、リスニングの練習を気軽に行うことができます。 単語学習に偏った学習習慣の改善と繰り返し音声を聞くことによるリスニング力の向上です。

## ユーザーの獲得について
まずテスト期間として、スクールに通う生徒の数名に実際に使用してもらい、フィードバックを得ます。それから全校配信して使用してもらう予定です。

## サービスの差別化ポイント・推しポイント
当スクールで扱うテキストは会社で開発したオリジナルテキストになります。そのテキストとリンクしたアプリは現時点では存在しません。

## 機能候補
＊ユーザーは会社で使用しているテキストと音声を併用して利用します。テキストは現時点で2冊のみ。

<リリースまでの機能>
・ユーザー登録（ユーザー名、ニックネーム、英検の級、Eメール、パスワード） 
・ログイン/ログアウト（一度ログインしたら、ログインの必要性なしにする） 
・音声の一覧画面から選択し、音声を再生することができる。 
・検索機能（ユーザーが選択したいテキストの音声を検索できる） 
・音声再生（リピート機能、音声速度変更、音量の調整、スキップ、一時停止、バックグラウンド再生可） 

<今後追加していく機能>
・いいね機能（自分のお気に入りのテキスト選択、聞きたい音源のいいね機能） 
・テキストの動画（Youtube）の閲覧可 
・学習記録を残す（学習したテキスト名、音声番号、学習した回数）
